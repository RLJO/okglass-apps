SALES TO PURCHASE ORDER / VICE VERSA

This module provides an extensive feature of creating a Purchase order from sales order and vice versa. While creating a sales order ,automatically an RFQ gets created. Similarly, when you create a purchase order the Sales Quotation gets generated.

HOW TO USE THIS MODULE

In sales Quotation you can see the Vendor field and an PO field which is an link to the purchase order. Confirm the Quotation after specifying the details.

After Confirming the Quotation,automatically an RFQ gets created and the Link to the PO can be seen in the related PO field. .

When you click on the PO link it will redirect you to the newly created RFQ.

Similarly, From Purchase you can create an Sales Quotation, As seen in the above image, The customer can be selected for creating the sales.

After confirmation, Sales Quotation gets generated and the link is generated in the SO field.

Again, by clicking on the SO link you will be redirected to the Sales Quotation generated.
