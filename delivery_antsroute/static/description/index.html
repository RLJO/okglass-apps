Odoo AntsRoute Delivery Connector

AntsRoute is a SaaS software for route planning and optimization which allows delivery companies to determine the best routes. If you use Odoo as your backend ERP and have an AntsRoute account to manage your orders and optimize routes, Odoo AntsRoute Delivery Connector is a must-have solution for your team. Using this integration, you can automate various operations to make your work easier, such as transferring delivery orders and customer data from Odoo to define the best delivery routes in AntsRoute, updating order status in Odoo.

  • Choose the delivery days.
  • Define warehouse default opening hours and warehouse default closing hours.

  • Determine the best delivery routes using AntsRoute.
  • Take into consideration the previous information defined in Odoo.
  • Take into consideration constraints defined in AntsRoute such as - management of time slots, working hours of drivers, lunch breaks, starting and finishing addresses.
  • Take into consideration the capacities of vehicles.

Add a new shipping method

Once you have activated the AntsRoute app, you need to create a new shipping method from the 'Inventory' module of your Odoo account. Check the box 'Use AntsRoute' and select or add the product associated with this shipping method.

Enter the AntsRoute API Key

After having checked the box 'Use AntsRoute', you access an 'AntsRoute' tab from which you can paste the API key retrieved from your AntsRoute account.

Select order destination in AntsRoute

Now choose the AntsRoute scheduling mode among the two options: 'In Organize' or 'In the bucket'.

Note : The bucket of orders allows you to add deliveries with a due date, i.e. a date before which the delivery must be completed. In contrast, when choosing 'In Organize', the orders are directly add in the routes on defined date.

Set up the time constraints from Odoo

From the 'AntsRoute' tab, define the following elements taking into consideration when scheduling orders and optimising routes: average loading time at the warehouse, average unloading time at customer's, delivery days, warehouse default opening hours, warehouse default closing hours, customer default delivery starting time, customer default delivery ending time.

Create capacities in AntsRoute

From you AntsRoute account, create the capacities 'Number', 'Weight' and 'Volume' (AntsRoute > Settings > Capacities > Add). Then, enter the carrying capacities for each vehicle (Resources > Vehicles > … > Edit > Capacities).

Add extra fields in AntsRoute

From your AntsRoute account, create the extra fields 'Price', 'Description' and 'Reference' (AntsRoute > Settings > Extra fields > Order/appointment > Add). For 'Display screen', select 'Create/update screen'.

Add new customer

When creating new quotation from the 'Sales' module of your Odoo account, you must add customer with name and address (Street, ZIP and City).

Note : You can select already existing customer.

Choose the shipping method

Once you have indicated the products to deliver, you must select the shipping method using AntsRoute by clicking on 'Add shipping'. Then, 'Confirm' the quotation.

Define the quantity to deliver

From 'Delivery' > 'Edit' > 'Detailed operations', edit the quantity indicated in 'Done' by entering the number of products to deliver.

Request carrier pickup

Click on the 'Request carrier pickup' button to send data to AntsRoute.

View order status from Odoo

From your Odoo account, you can access information about the order status imported from AntsRoute by going to Orders > Delivery > AntsRoute Information. You just have to click on the 'Get Delivery Information' button.
